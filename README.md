# Проект Mesto REST API
 * можно удалять/добавлять карточки с фотографиями
 * лайкать/снимать лайк, счетчик лайков меняется
 * менять аватар
 * менять информацию о пользователе
 
Настроен сервер на Express
  
##Cтэк
* Node.js
* Express
* MongoDB
